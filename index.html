Example 2: Highlighting the First List Item

In this example, the first city in the list will be highlighted when the button is clicked. It shows how to traverse the DOM and apply a style to a specific element.

  • Frankfurt
  • Hamburg
  • Munich

Example 7: Event Delegation

In this example, clicking any item in the list will trigger an alert displaying the name of the tea clicked. This is done using event delegation.

  • Green Tea
  • Oolong Tea
  • Black Tea

Click on any tea to select it.

            
            
            
                <h2>Example 7: Event Delegation</h2>
                <p>In this example, clicking any item in the list will trigger an alert displaying the name of the tea clicked. This is done using event delegation.</p>
                <ul id="teaList">
                    <li class="teaItem">Green Tea</li>
                    <li class="teaItem">Oolong Tea</li>
                    <li class="teaItem">Black Tea</li>
                </ul>
                <p>Click on any tea to select it.</p>                           
            
        
            

    document.getElementById("teaList").addEventListener("click", function (event) {
        if (event.target && event.target.matches(".teaItem")) {
            alert("you selected " + event.target.textContent);
        }
    });

Example 8: Form Handling

This example demonstrates how to capture form input and display it below the form after submission.

            
            
            
                <h2>Example 8: Form Handling</h2>
                <p>This example demonstrates how to capture form input and display it below the form after submission.</p>
                <form id="feedbackform">
                    <label for="feedbackInput">Your Feedback:</label>
                    <input type="text" id="feedbackInput" placeholder="Enter feedback">
                    <button type="submit">Submit Feedback</button>
                </form>
                <p id="feedbackDisplay"></p>
                             
            
        
            

    document.getElementById("feedbackform").addEventListener("submit", function(event){
        event.preventDefault();
        let feedback = document.getElementById("feedbackInput").value;
        document.getElementById("feedbackDisplay").textContent = `feedback is : ${feedback}`;
    })
